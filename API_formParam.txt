Short Definition (Interview-Perfect)

formParam is used to send form-encoded request parameters (application/x-www-form-urlencoded) in the HTTP request body, typically for HTML form submissions and OAuth token APIs.

ğŸ”¹ When should you use formParam?

Use formParam ONLY when the API expects data in:

Content-Type: application/x-www-form-urlencoded


This is NOT JSON.
