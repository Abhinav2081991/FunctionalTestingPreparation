Difference Between POM and Page Factory Pattern (Selenium)
1Ô∏è‚É£ What is Page Object Model (POM)?
Definition

Page Object Model is a design pattern where each web page of the application is represented as a separate Java class, and all the page elements and actions are defined inside that class.

üëâ POM is a design concept, not a Selenium feature.

Example ‚Äì POM (Without PageFactory)
public class LoginPage {

    WebDriver driver;

    By username = By.id("username");
    By password = By.id("password");
    By loginBtn = By.id("login");

    public LoginPage(WebDriver driver) {
        this.driver = driver;
    }

    public void login(String user, String pass) {
        driver.findElement(username).sendKeys(user);
        driver.findElement(password).sendKeys(pass);
        driver.findElement(loginBtn).click();
    }
}


‚úî Clean separation
‚úî Better maintainability
‚úî Uses By locators

2Ô∏è‚É£ What is Page Factory?
Definition
Page Factory is a Selenium support library that implements POM internally and provides:
Lazy element initialization
@FindBy annotations
Cleaner syntax

üëâ Page Factory is a Selenium implementation of POM

Example ‚Äì Page Factory
public class LoginPage {

    WebDriver driver;

    @FindBy(id = "username")
    WebElement username;

    @FindBy(id = "password")
    WebElement password;

    @FindBy(id = "login")
    WebElement loginBtn;

    public LoginPage(WebDriver driver) {
        this.driver = driver;
        PageFactory.initElements(driver, this);
    }

    public void login(String user, String pass) {
        username.sendKeys(user);
        password.sendKeys(pass);
        loginBtn.click();
    }
}


‚úî Less boilerplate
‚úî Lazy loading
‚úî Readable code

5Ô∏è‚É£ Which One Should You Use?

Use POM (Without PageFactory) when:
	You want full control over element handling
	Handling highly dynamic elements
	Avoid stale element issues
	Framework-level abstraction

Use Page Factory when:
	Application UI is stable
	Want cleaner, readable code
	Faster development
	Smaller projects
